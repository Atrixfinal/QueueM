# 🏥 Hospital Queue Management System

A simple and efficient Queue Management System designed to streamline patient flow in hospitals. This project helps manage patient queues, reduce waiting time, and improve overall operational efficiency.

---

## 🚀 Features

* 🧑‍⚕️ Patient Registration
* 📋 Queue Generation & Management
* ⏱️ Real-time Queue Tracking
* 🔔 Token/Number System
* 📊 Basic Dashboard for Monitoring
* 🗂️ Organized Data Handling

## 💡 How It Works

1. Patient registers at the reception
2. System generates a unique token number
3. Patient is added to the queue
4. Doctor/Admin calls next patient
5. Queue updates automatically
